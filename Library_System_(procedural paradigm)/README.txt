# Library Management System (Procedural)

A console-based Library Management System developed in C++ using the Procedural Programming paradigm. The system allows users to manage books efficiently through a menu-driven interface.

## Features

- Add new books.
- Remove books by ID.
- Update book information.
- Search for books by ID.
- Borrow books.
- Return borrowed books.
- Display a specific book's information.
- Display all books.
- Automatic library capacity expansion using dynamic memory allocation.
- Welcome screen with ASCII art.

## Concepts Used

- Procedural Programming
- Dynamic Memory Allocation
- Pointers
- Function Templates
- Arrays
- File Organization
- Input Validation
- Searching Algorithms

## Technologies

- C++
- Visual Studio
- Windows Console API

## Project Structure

- Dynamic memory management
- Book management
- Search functionality
- Borrow/Return system
- Library expansion
- Console user interface

## How to Run

1. Open the solution (`.sln`) file in Visual Studio.
2. Build the project.
3. Run the application.
4. Enter the initial library capacity.
5. Use the menu to manage books.
