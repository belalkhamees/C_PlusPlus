Difference between composition and inheritance is:

1- composition:
-shows the (has a) relationship between classes. 
-it happens when the class has an object attribute from another class. 
-if the owner object of a class is destroyed ,the contained object of the other 
class is destroyed too.

2- inheritance:

-shows the (is a) relationship between base class and sub classes.
-its permits code reusability instead of repetition of the same attributes in each class.  
-the subclass inherits all the capabilities of the base class but can add its own features
(local attribute) , and base class is unchanged by this process.
